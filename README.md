# Printii Site (Next.js + Tailwind)

## Run locally
```bash
npm install
npm run dev
# open http://localhost:3000
```

## Deploy to Vercel
- Import this repo at https://vercel.com/new
- Add domain `printii.co.uk` in Project → Settings → Domains
- Done

## Notes
- Logo is in /public as printii-logo.png (or printii-logo.svg if placeholder)
- Contact form currently opens an email draft to `info@printi` as requested.
- To change, edit `app/PrintiiLanding.tsx`.
